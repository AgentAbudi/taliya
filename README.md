# Taliya — تالية

> A simple dhikr companion that helps you remember Allah — one tap at a time.

## Features

- **Tap Counter** — Count Astaghfirullah, Alhamdulillah, SubhanAllah, and Allahu Akbar
- **Daily Progress** — Set goals (33, 100, 500, 1000) and track progress
- **Multiple Dhikr Modes** — Easily switch between forms of remembrance
- **PWA Ready** — Install on your phone for instant access
- **Calm Design** — Soft cream & emerald green, distraction-free

## Tech Stack

- Next.js 15 + React 19
- Tailwind CSS 4
- TypeScript
- PWA with Web App Manifest

## Getting Started

npm install
npm run dev

## Meaning

Taliya reflects the act of reciting, remembering, and staying connected.
